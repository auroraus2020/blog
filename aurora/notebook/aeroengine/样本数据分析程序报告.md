### 一、程序介绍

   在科学研究的过程中，我们往往要处理大量的数据，分析数据的特征和分布情况等，本文中所介绍的程序就是为了处理这些数据而编写的，为了方便其他人的使用，我把很多主要的功能都封装成了一些函数，在自己使用的时候直接调用就可以了。在这里也顺便提一句，希望大家也都能在组内分享自己一些算例的源程序，以促进大家功能进步。

  在本程序的编写中，用到了概率论与数理统计、数值分析的知识，在后面出现的地方会详细介绍。

  本程序的作用主要有：

- 计算样本数据的**概率密度**，并求出其对应的拟合函数
- 计算样本数据的**分布函数**，并求出其对应的拟合函数
- 对于样本数据分析后根据样本数据建立相应的统计模型，通过拉丁超立方采样发进行样本的抽取
- 通过建立的统计模型，计算数学期望
- 求解定积分
- 求解拟合函数
- 求解任意函数的反函数

```python
'''求样本数据的概率密度函数
参数x：样本集，是一个list
参数ratio：计算区间大小控制参数
返回值为计算区间中心点和其区间对应的概率密度
'''
def Pdf(x,ratio=10):
    xsize=len(x)
    xmin=min(x)
    SectionSize=int(np.floor((len(x)/ratio)))+1  #区间个数
    dx=float(max(x)-min(x))/SectionSize  #区间长度
    value=[]
    pdf=[]
    for i in range(SectionSize-1):
        k=0
        for j in x:
            if j>=xmin+i*dx and j<xmin+(i+1)*dx:
                k+=1
        pdf.append(k/(xsize*dx))
        value.append(xmin+(i+0.5)*dx)
    return np.array(value),np.array(pdf)
```

##### 2.3、拉丁超立方采样发进行统计模型样本的抽取

该函数的功能是通过样本分布函数的拟合函数（统计模型）来抽取任意多数量的样本，以更加真实的反映实际样本。关于拉丁超立方采样，请参考前面的文章，有专门的介绍。

```python
'''LHS采样
参数down：采样下限值
参数up：采样上限值
参数function:函数表达式（字符串形式，其中的因变量用x表示）
参数N：最大迭代次数
参数esp：返回值的误差
返回值：函数值对应的x值（list）
'''
def LHS_Sample(down,up,scale,function,SampleNum):
    low=Eval(np.array([down])/scale,function)
    high=Eval(np.array([up])/scale,function)
    ds=Linspace(low,high,SampleNum)
    ds=np.array(ds)  #Pdf为累计概率分布曲线上均匀分层取的点
    Sample=InverseFunction(ds,down/scale,up/scale,function)
    #Sample=Eval(ds,inversefunction)
    np.random.shuffle(Sample)
    return Sample*scale
```

'''由概率密度函数求数学期望

def ExpectedValue(x,function):
    x=np.array(x)

##### 2.6、求解拟合函数

下面的两个函数用来计算拟合函数，两者的区别在于前者返回值为多项式拟合函数的系数参数，后者返回值为拟合函数的表达式（字符串形式）。

**为什么要使用多项式拟合函数？**

答案：因为根据泰勒公式的推论麦克劳林公式可知，可以实用多项式函数拟合任何函数，并且随着多项式幂次n的增大，拟合函数和实际函数的误差越来越小。

```python
'''计算多项式拟合函数的系数参数
参数x:因变量
参数y:自变量
参数n：拟合函数中因变量的最高幂
返回值：多项式前的系数
'''
def PolyFit(x,y,n=5):
    x=np.array(x)
    y=np.array(y)
    coff=np.polyfit(x,y,n)
    return coff


'''计算多项式拟合函数的表达式
参数x:自变量
参数y:因变量
参数n：拟合函数中因变量的最高幂
返回值：拟合函数表达式(字符串)
'''
def FitFunction(x,y,n=5):
    coff=PolyFit(x,y,n=n)
    f=''
    for i in range(n):
        f+=str(coff[i])+'*x^'+str(n-i)+' +'
    f+=str(coff[n])
    return f
```

##### 2.7、求解任意函数的反函数

该函数和主要功能是计算上面提到的多项式拟合函数的反函数，关于该反函数的求解，可谓大费周折，最先考虑也用拟合函数的方法来拟合反函数，这种方法可行，但是其效果不好，通过多项式拟合得到的反函数LHS抽样得到的样本分布差强人意，与实际分布规律相差较大。后面找数学系的同学请教，该同学提供了切比雪夫多项式和幂级数求和的方法，但是这两种方法比较复杂，在这里不适用。

最后冥思苦想想到了数值分析里面的求根算法——**牛顿二分法**，在这里把求反函数的问题转化为求根的问题（由于分布函数是单调不减函数，所有对应方程的根在不同函数值时仅有一个）。问题顺利解决，通过该算法得到的样本分布符合实际样本的分布规律。

**转换思路如下所示：**

![img](https://auroraus-1258416167.cos.ap-beijing.myqcloud.com/NoteBook/Note/%E6%A0%B7%E6%9C%AC%E6%95%B0%E6%8D%AE%E5%88%86%E6%9E%90%E7%A8%8B%E5%BA%8F%E6%8A%A5%E5%91%8A/wps6.png)

```python
'''求反函数
参数x：要求反的函数值(list)
参数low：函数值对应的x值下限
参数high：函数值对应的x值上限
参数function:函数表达式（字符串形式，其中的因变量用x表示）
参数N：最大迭代次数
参数esp：返回值的误差
返回值：函数值对应的x值（list）
'''
def InverseFunction(x,low,high,function,N=1000,eps=1e-10):
    result=x
    for i in range(len(x)):
        if i==0:
            a=low
            b=high
        else:
            a=a
            b=high
        c=0
        step=0
        while step<N:
            step+=1
            f1=Eval(a,function)
            c=(a+b)/2
            f3=Eval(c,function)
            f2=Eval(b,function)
            
            err=abs(b-a)
            if err<eps:
                result[i]=c
                break
            if x[i]>f1 and x[i]<f3:
                a=a
                b=c
            elif x[i]>f3 and x[i]<f2:
                a=c
                b=b
            elif x[i]==f1:
                result[i]=a
                break
            elif x[i]==f2:
                result[i]=b
                break
            elif x[i]==f3:
                result[i]=c
                break
    return result
```

##### 2.8、样本数据读取函数

该函数为一个辅助函数，用来从txt文件中读取样本数据，需要注意的是，所读取的原始数据分隔符必须为英文逗号、空格、制表符或者换行符中的其中一种。为了减小计算误差计算和增加计算效率，这里参对读取的数据进行了归一化处理（也就是将原始数据缩放到1.0附近）。

```python
'''读取样本数据函数
参数path：样本数据文件路径
返回值为一个list,和归一化系数
'''
def ReadData(path):
    with open(path,'r') as f:
        data=f.read().replace('\n\n','\n')
        f.close()
    if " " in data:
        x=data.split(' ')
    elif "," in data:
        x=data.split(',')
    elif "\t" in data:
        x=data.split('\t')
    elif "\n" in data:
        x=data.split('\n')
    else:
        pass
    x=[float(i) for i in x]   #全部转换为浮点值
    xmax=max(x)
    xmin=min(x)
    if abs(xmax)>1.0:
        scale=abs(xmax)+abs(xmin)
        x=[float(i/scale) for i in x]   #全部转换为浮点值
    else:
        scale=1.0
    return np.array(x),scale
```

刚开始手头暂时没有较好的样本数据源，找来找去最后在某考研群里发现了北航2018年考研录取学生成绩总表，所以在这里选择北航2018级研究生的考研成绩作为样本数据。抽取其中一部分学生（1436个样本数据）的政治考研成绩作为所研究的样本。[点击这里获取样本数据](https://auroraus-1258416167.cos.ap-beijing.myqcloud.com/NoteBook/Note/%E6%A0%B7%E6%9C%AC%E6%95%B0%E6%8D%AE%E5%88%86%E6%9E%90%E7%A8%8B%E5%BA%8F%E6%8A%A5%E5%91%8A/%E5%8C%97%E8%88%AA%E9%83%A8%E5%88%86%E5%90%8C%E5%AD%A6%E8%80%83%E7%A0%94%E6%94%BF%E6%B2%BB%E6%88%90%E7%BB%A9.txt)

**计算程序如下所示**

```python
x,scale=ReadData('北航部分同学考研政治成绩.txt')  #读取原始数据
value,pdf=Pdf(scale*x,25) #计算概率密度
value,ds=DS(scale*x,25) #计算分布函数
    
pylab.figure(1)#绘制样本分布直方图
pylab.hist(x = x*scale, bins = 40, color = 'steelblue', edgecolor = 'black' )

plot(2,value,pdf,color='b',title='PDF')  #绘制出计算得到的概率密度
plot(2,value,Eval(scale*value,FitFunction(scale*value,pdf,n=10)),color='r--',title='PDF') #绘制出拟合后的概率密度函数曲线

plot(3,value,ds,color='b',title='DS')   #绘制出计算得到的分布密度
plot(3,value,Eval(scale*value,FitFunction(scale*value,ds,n=10)),color='r--',title='DS')   #绘制出拟合后的分布函数曲线

pylab.figure(4)
data=LHS_Sample(45,75,scale,FitFunction(value/scale,ds,n=12),1000)#LHS抽样
pylab.hist(x = data, bins = 100, color = 'steelblue', edgecolor = 'black' )#抽样样本直方图

Expectation=ExpectedValue(x*scale,FitFunction(value,pdf,n=10))#计算数学期望
print('该分布的数学期望为：',Expectation)
```
