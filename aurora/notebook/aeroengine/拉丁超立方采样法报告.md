### 一、拉丁超立方采样法（Latin hypercube sampling，LHS）简介

#### 1.1、LHS方法介绍

  LHS方法是由M. D. Mc Kay、R. J .Beckman等人于1979年提出的，是一种受约束的均匀的抽样法,拉丁超立方抽样法的目的是要保证所有的采样区域都能被采样点覆盖住。拉丁超立方抽样根据各输人参数的分布函数和定义域范围，采取等概率分层抽样产生各参数的随机数样本。其首先确定模拟次数N，然后根据模拟次数将变量的概率分布函数等分成N个互不重叠的子区间，最后在每个子区间内分别进行独立的**等概率抽样**。

- 将上一步中抽出的样本随机组合，从而形成N个符合该分布函数的随机样本

#### 1.3、LHS方法的特点

- 在样本数较少时，可以避免样本聚集的问题
- 对概率低的样本也会强制抽样，避免了样本数较少时计算结果失真的问题
- 相比于蒙特卡洛法，可以减少计算步数
- 知道某个随机变量的概率密度函数或分布函数，即可产生满足其分布规律的随机数

```python
# -*- coding: utf-8 -*-

import random
from scipy.stats import norm,uniform
import numpy as np
import pylab

'''均匀分布LHS抽样函数'''
def LHS_Uniform(down,up,SampleNum):
    Pdf=[]
    for i in range(SampleNum):
        Pdf.append((i+0.5)/SampleNum)
    Pdf=np.array(Pdf)  #Pdf为累计概率分布曲线上均匀分层取的点
    Sample=uniform(down,up).ppf(Pdf)
    np.random.shuffle(Sample) #打乱随机排序
    np.random.shuffle(Sample)
    return Sample

'''正太分布分布LHS抽样函数'''
def LHS_Norm(mean,sigma,SampleNum,down=None,up=None,method=''):
    Pdf=[]
    if method=='sigma':
        low=norm(mean,sigma).cdf(mean-down*sigma)
        high=norm(mean,sigma).cdf(mean+up*sigma)
        for i in range(SampleNum):
            prob=(i+random.random())/SampleNum
            if prob>=low and prob<=high:
                Pdf.append(prob)
    elif method=='value':
        low=norm(mean,sigma).cdf(down)
        high=norm(mean,sigma).cdf(up)
        for i in range(SampleNum):
            prob=(i+random.random())/SampleNum
            if prob>=low and prob<=high:
                Pdf.append(prob)
    else:
        for i in range(SampleNum):
            Pdf.append((i+random.random())/SampleNum)
        
    Pdf=np.array(Pdf)  #Pdf为累计概率分布曲线上均匀分层取的点
    Sample=norm(mean,sigma).ppf(Pdf)
    np.random.shuffle(Sample)
    return Sample,Pdf
```

##### 3.1.1 、组合偏差概率计算

**问题描述：**某产品由八个零件堆叠组成。也就是说，这八个零件的厚度总和，等于该产品的厚度。如下图所示。现已知该产品的厚度必须控制在27mm以内，但是每个零件的厚度都有一定的概率的偏差，厚度可能会超出误差。请问有多大的概率，产品的厚度会超出27mm？或者说该产品的合格率多大？下表为每个零件的详细情况。

![image-20200716123050569](https://auroraus-1258416167.cos.ap-beijing.myqcloud.com/NoteBook/Note/%E6%8B%89%E4%B8%81%E8%B6%85%E7%AB%8B%E6%96%B9%E9%87%87%E6%A0%B7%E6%B3%95%E6%8A%A5%E5%91%8A/image-20200716123050569.png)

**求解思路：**利用蒙特卡罗方法的思想，取一定数量的随机样本（例如1000000个），每个样本有8个值，对应8个零件各自的厚度。假设每个零件的厚度都满足正态分布，最后将这些零件厚度加和就是样本的厚度。从而可以求出该产品的良品率（或者称合格率）。这种结果的原因是零件中的八个零件各自的尺寸是随机打乱的并加和的，在这里LHS算法的优势被削弱了。
