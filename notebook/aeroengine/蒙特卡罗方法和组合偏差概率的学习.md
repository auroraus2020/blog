## 一、什么是蒙特卡罗法

​		蒙特卡罗方法是一种计算方法。原理是通过大量随机样本，去了解一个系统，进而得到所要计算的值。它非常强大和灵活，又相当简单易懂，很容易实现。对于许多问题来说，它往往是最简单的计算方法，有时甚至是唯一可行的方法。而且它特别适用于一些解析法难以求解甚至不可能求解的问题，因而在装备效能评估中具有重要地位。

#### 1.1 基本思路

​		蒙特卡洛法的基本思想是：为了求解问题，首先建立一个概率模型或随机过程，使它的参数或数字特征等于问题的解：然后通过对模型或过程的观察或抽样试验来计算这些参数或数字特征，最后给出所求解的近似值。解的精确度用估计值的标准误差来表示。蒙特卡洛法的主要理论基础是概率统计理论，主要手段是随机抽样、统计试验。用蒙特卡洛法求解实际问题的基本步骤为：

- 根据实际问题的特点．构造简单而又便于实现的概率统计模型．使所求的解恰好是所求问题的概率分布或数学期望；

- 给出模型中各种不同分布[随机变量](https://baike.baidu.com/item/随机变量)的抽样方法；

- 统计处理模拟结果，给出问题解的统计估计值和精度估计值。

#### 1.2 优缺点

蒙特卡罗法的最大优点是:

- 方法的[误差](https://baike.baidu.com/item/误差)与问题的维数无关。

- 对于具有统计性质问题可以直接进行解决。

- 对于连续性的问题不必进行离散化处理

蒙特卡罗法的缺点则是:

- 对于确定性问题需要转化成随机性问题。

- 误差是概率误差。

- 通常需要较多的计算步数N.

#### 1.3 使用步骤

（1）对每一项活动，输入最小、最大和最可能估计数据（注意这里不是三点估算），并根据提出的问题构造或选择一个简单、适用的概率分布模型，使问题的解对应于该模型中随机变量的某些特征（如概率、均值和方差等），这些特征都可以通过模拟出的概率分布图得到。

（2）根据模型中各个随机变量的分布，利用给定的某种规则，在计算机上快速实施充分大量的随机抽样。

（3）对随机抽样的数据进行必要的数学计算，统计分析模拟试验结果，给出问题的概率解以及解的精度估计，即最小值、最大值以及数学期望值和单位标准偏差。

（4）按照所建立的模型进行仿真试验、计算，求出问题的随机解。

（5）根据求出的统计学处理数据，让计算机自动生成概率分布图，通常为正态分布图。

（6）根据概率分布图读出所需信息，如某项目成本200万情况下的完工概率，或确保70%完工概率时需要的成本等。

## 二、应用举例

​	以下通过三个实例来说明蒙特卡罗方法。分别是：**圆周率Pi的计算、定积分的计算、组合概率的计算**。分别如下所示：

### 2.1 圆周率Pi的计算

#### 2.1.1 问题描述

​	圆周率的计算可以转化为面积比的计算问题，如下图所示，正方形内切圆的面积和正方形的面积之比为Pi/4，所以只要想办法求出圆的面积，就可以通过面积比求出圆周率Pi的值。

![](https://auroraus-1258416167.cos.ap-beijing.myqcloud.com/ESRG/Report/%E8%92%99%E7%89%B9%E5%8D%A1%E6%B4%9B%E5%A4%AB%E5%AD%A6%E4%B9%A0/pi_count1.jpg)

#### 2.1.2 求解思路

​	利用蒙特卡罗方法的思想，在这个正方形内部，随机产生一定数目的点（例如100000个），计算它们与中心点的距离，判断是否落在圆的内部。其中落在圆的内部的点数量/总的点数=Pi/4，从而求出Pi的数值。

![](https://auroraus-1258416167.cos.ap-beijing.myqcloud.com/ESRG/Report/%E8%92%99%E7%89%B9%E5%8D%A1%E6%B4%9B%E5%A4%AB%E5%AD%A6%E4%B9%A0/pi_count2.jpg)

#### 2.1.3 计算代码

```python
# -*- coding: utf-8 -*-
"""
Created on Tue Jun  9 15:08:44 2020

@author: zhangfan
"""
import numpy

Pi=numpy.pi
SampleNum=1000000
point=numpy.random.uniform(0,2,2*SampleNum)
point=point.reshape(-1,2)
x=point[:,0]
y=point[:,1]
Distance=numpy.sqrt((x-1.0)**2+(y-1.0)**2)
Cnum=0
for i in Distance:
    if i<=1.0:
        Cnum+=1
print('圆周率真实值为：',Pi)
print('蒙特卡洛法计算得到的值为：',4*Cnum/SampleNum)
print('蒙特卡洛法计算得到的值与真实值相差为：',abs(4*Cnum/SampleNum-Pi)/Pi*100,'%')
```

#### 2.1.4 计算结果

​	设置1000000个随机的点，由上面的python程序计算得到Pi的数值为：Pi=3.140604，上述方法求得值与真实值相差0.0315%。如下图所示。

![](https://auroraus-1258416167.cos.ap-beijing.myqcloud.com/ESRG/Report/%E8%92%99%E7%89%B9%E5%8D%A1%E6%B4%9B%E5%A4%AB%E5%AD%A6%E4%B9%A0/Pi.png)



### 2.2 定积分的计算

#### 2.2.1 问题描述

​	用蒙特卡洛法求解定积分y=x^2在 [0, 1] 区间的积分，也就是下图所示的红色部分面积

![](https://auroraus-1258416167.cos.ap-beijing.myqcloud.com/ESRG/Report/%E8%92%99%E7%89%B9%E5%8D%A1%E6%B4%9B%E5%A4%AB%E5%AD%A6%E4%B9%A0/int_count2.gif)

#### 2.2.2 求解思路

​	上面求解Pi的方法加以推广就可以求解任何一个积分的值了（例如求不出原函数的那些积分等），求解思想如下图所示

![](https://auroraus-1258416167.cos.ap-beijing.myqcloud.com/ESRG/Report/%E8%92%99%E7%89%B9%E5%8D%A1%E6%B4%9B%E5%A4%AB%E5%AD%A6%E4%B9%A0/int_count1.gif)

#### 2.2.3 计算代码

```python
# -*- coding: utf-8 -*-
"""
Created on Tue Jun  9 15:30:44 2020

@author: zhangfan
"""
import numpy

SampleNum=1000000
point=numpy.random.uniform(0,1,2*SampleNum)
point=point.reshape(-1,2)
x=point[:,0]
y=point[:,1]
num=0
for i in range(len(x)):
    if y[i]<x[i]*x[i]:
        num+=1
print('该定积分的真实面积为：',1/3)
print('蒙特卡洛法计算得到的该定积分面积为：',num/SampleNum)
print('蒙特卡洛法计算得到的值与真实值相差为：',abs(num/SampleNum-1/3)*3*100,'%')
```

#### 2.2.4 计算结果

设置1000000个随机的点，由上面的python程序计算得到定积分的数值为：0.333084，上述方法求得值与真实值相差0.0748%。如下图所示。

### 2.3 组合概率的计算

#### 2.3.1 问题描述

​	某产品由八个零件堆叠组成。也就是说，这八个零件的厚度总和，等于该产品的厚度。如下图所示。
![](https://auroraus-1258416167.cos.ap-beijing.myqcloud.com/ESRG/Report/%E8%92%99%E7%89%B9%E5%8D%A1%E6%B4%9B%E5%A4%AB%E5%AD%A6%E4%B9%A0/com_prob1.jpg)

​	现已知该产品的厚度必须控制在27mm以内，但是每个零件的厚度都有一定的概率的偏差，厚度可能会超出误差。请问有多大的概率，产品的厚度会超出27mm？或者说该产品的合格率多大？下表为每个零件的详细情况。

![](https://auroraus-1258416167.cos.ap-beijing.myqcloud.com/ESRG/Report/%E8%92%99%E7%89%B9%E5%8D%A1%E6%B4%9B%E5%A4%AB%E5%AD%A6%E4%B9%A0/com_prob2.png)

#### 2.3.2 求解思路

​	利用蒙特卡罗方法的思想，取一定数量的随机样本（例如1000000个），每个样本有8个值，对应8个零件各自的厚度。假设每个零件的厚度都满足正态分布，最后将这些零件厚度加和就是样本的厚度。从而可以求出该产品的良品率（或者称合格率）。

![](https://auroraus-1258416167.cos.ap-beijing.myqcloud.com/ESRG/Report/%E8%92%99%E7%89%B9%E5%8D%A1%E6%B4%9B%E5%A4%AB%E5%AD%A6%E4%B9%A0/normal.png)

#### 2.3.3 计算代码

```python
# -*- coding: utf-8 -*-
"""
Created on Tue Jun  9 16:35:44 2020

@author: zhangfan
"""
import numpy as np
import pylab

SampleNum=100000

#正态分布的概率密度函数。可以理解成 x 是 mu（均值）和 sigma（标准差）的函数
def normfun(x,mu,sigma):
    pdf = np.exp(-((x - mu)**2)/(2*sigma**2)) / (sigma * np.sqrt(2*np.pi))
    return pdf/100

aa=np.random.normal(2,0.0455,SampleNum) #正太分布的随机数
a=normfun(aa,2,0.0455)  #求概率密度

bb=0.5
b=0

cc=np.random.normal(4.575,0.035667,SampleNum)
c=normfun(cc,4.575,0.035667)

dd=np.random.normal(3,0.0455,SampleNum)
d=normfun(dd,3,0.0455)

ee=np.random.normal(1,0.033867,SampleNum)
e=normfun(ee,1,0.033867)

ff=np.random.normal(13,0.05,SampleNum)
f=normfun(ff,13,0.05)

gg=0.5
g=0

hh=np.random.normal(2,0.0455,SampleNum)
h=normfun(hh,2,0.0455)

k=0
t=a+b+c+d+e+f+g+h
tt=aa+bb+cc+dd+ee+ff+gg+hh

for i in tt:
    if i<=27:
        k+=1

print('产品厚度小于27mm的概率(即合格率)为：',k/SampleNum*100,'%')
pylab.plot(tt,t,'b.')
pylab.grid(True)
pylab.title("Height Distribution")
pylab.xlabel('Total Height')
pylab.ylabel('Probability density')
```

​	取100000个随机样本，最后算得产品的合格率为：99.999%，结果和产品厚度分布图如下所示：
